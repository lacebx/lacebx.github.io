## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Node.js and npm for local development setup

### Installation

1. Clone the repository:
```bash
git clone https://github.com/lacebx/lacebx.github.io.git
cd lacebx.github.io
```

2. Open `index.html` in your web browser to view the portfolio

Or, if using a local development server:
```bash
npm install
npm start
```
